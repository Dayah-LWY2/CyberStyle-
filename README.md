# CyberStyle
A first website development assignment in 2024 involved creating an e-commerce website hosted on Render with basic file management. The project was developed using HTML, CSS, and JavaScript for the frontend, and Node.js, Express.js, API architecture, and JSON files for data management.

The website prototype was created by Azlina Lee. Meanwhile, product designs were developed by Puteri Nurelisah, Rahmah, Hakeem, and Ehsan.
